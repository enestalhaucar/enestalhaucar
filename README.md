### 🏗️ Currently building — Vegatra

> **Automation, built for e-commerce.**

**[Vegatra](#)** is an AI customer-operations platform for e-commerce sellers. It connects every channel a seller actually uses — marketplaces (Trendyol, Hepsiburada, N11, Çiçeksepeti), their own store (Shopify, İkas, Ticimax) and social DMs (WhatsApp, Instagram) — into one brain.

The difference: it doesn't just *answer* customer questions, it **takes action** — processing returns, exchanges, refunds and shipping updates end to end, with the seller's own rules. Reply tools stop at the question; Vegatra finishes the job.
